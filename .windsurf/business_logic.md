# Local SEO Optimizer - Business Logic & Value Proposition

> 이 문서는 프로젝트의 핵심 비즈니스 로직과 가치 제안을 정의한다.
> 모든 코드(프론트엔드 텍스트, 백엔드 로직, 리포트 생성)는 아래 원칙을 따라야 한다.

---

## 🎯 1. Target Market

**미국 로컬 비즈니스**
- 식당 (Restaurants)
- 스파 & 뷰티 (Spas, Salons)
- 클리닉 (Dental, Medical Clinics)
- 자동차 서비스 (Auto Repair, Car Wash)
- 기타 지역 서비스업

---

## 💎 2. Core Value

### "Time Saving" & "Revenue Growth (via Google Maps)"

- 사장님의 시간을 아껴준다
- 구글 지도를 통한 매출 증가를 실현한다
- 복잡한 마케팅을 자동화한다

---

## 📊 3. Key Metrics (KPI)

> ❌ 좋아요(Likes)가 아니다!
> ✅ **전화(Calls), 길찾기(Directions), 예약(Bookings)**이 핵심 지표다.

### 대시보드 표시 우선순위

```
1위: 📞 이번 주 전화 온 횟수 (Calls)
2위: 🗺️ 이번 주 길찾기 수 (Directions)
3위: 📅 이번 주 예약 수 (Bookings)
4위: ⭐ 새 리뷰 수 (New Reviews)
5위: 👁️ 프로필 조회수 (Profile Views)
```

### 리포트 필수 포함 항목
- 전화/길찾기/예약 수치 변화
- 경쟁사 대비 순위
- 예상 매출 기여도

---

## 🎨 4. Content Strategy

### One-Click Magic

> 사용자는 마케팅 전문가가 아니다.
> 복잡한 설정 없이 **'버튼 하나'**로 고품질 콘텐츠가 나와야 한다.

### UX 원칙: 객관식(Multiple Choice) Only

**❌ 나쁜 UX:**
```
"어떤 글을 써드릴까요?" [입력창]
```

**✅ 좋은 UX:**
```
"이번 주 프로모션 주제를 골라주세요:"

🌧️ 비 오는 날 따뜻한 스페셜 메뉴
🎃 할로윈 맞이 10% 할인
📅 주말 한정 가족 세트 홍보
🆕 신메뉴 출시 알림
⭐ 고객 리뷰 감사 포스트

[자동 생성하기] 버튼
```

### 콘텐츠 주제 자동 추천
- 날씨 기반 (비, 눈, 더위)
- 시즌/이벤트 기반 (할로윈, 크리스마스, 발렌타인)
- 요일 기반 (주말 특가, 월요일 블루스)
- 리뷰 기반 (최근 좋은 리뷰 활용)

### 1순위: GBP API 핵심 기능

1. **리뷰 자동 수집** (Review Collection)
   - 새 리뷰 알림
   - AI 답변 초안 생성
   - 원클릭 답변 게시

2. **포스트 업로드** (Post Publishing)

   - 새 질문 알림
   - AI 답변 추천
   - 자동 답변 게시

## 🔥 10. Conversion Funnel (전환 퍼널)

### 최종 Funnel 구조

```
(1) 회원가입
        ↓
(2) 상호명 + 주소 입력
        ↓
(3) AI 검색 & 분석
        ↓
(4) 상태 진단 + 개선 여지 설명
        ↓
(5) AI가 만들어주는 해결책 제시  ← 전환율 결정 핵심
        ↓
(6) 결제 CTA → 구독 시작 → 대시보드 진입
```

### 해결책 제시 3단계 레시피

```
⭐ 해결책 = Local SEO Optimizer
"Google Maps에서 더 많은 고객을 자동으로 데려오는 AI 엔진"

🔧 Step 1. 자동 콘텐츠 생성
   - 매주 AI가 업종·지역에 맞는 포스트 주제 추천
   - 클릭 한 번으로 이미지·본문 자동 생성
   - 승인하면 자동 업로드
   👉 활동성 증가 → Google Maps 노출 상승

📝 Step 2. 리뷰 관리 자동화
   - 새 리뷰 발생 → 즉시 알림
   - AI가 3가지 답변 초안 제시
   - 버튼 클릭 → 자동 게시
   👉 평점 유지 → 신뢰도 상승 → 고객 방문 증가

📈 Step 3. 전화/길찾기 증가 추적
   - 이번 주 Calls / Directions / Bookings 자동 수집
   - 경쟁사 대비 내 위치 분석
   - "다음 주 이렇게 하세요" 맞춤 추천
   👉 사장님이 직접 분석할 필요 없음 → 매출에 집중
```

## 🧩 11. Core Services (핵심 기능)

모든 유료 플랜에 기본 포함되는 4가지 핵심 기능:

### ⭐ CORE 1. Google Maps SEO 자동화

| 기능 | 설명 |
|------|------|
| Google Posts 자동 생성 | AI가 업종/지역 맞춤 포스트 생성 |
| 자동 업로드 | 승인 후 GBP에 자동 게시 |
| 리뷰 자동 수집 | 새 리뷰 실시간 감지 |
| AI 리뷰 답변 | 3가지 답변 초안 → 원클릭 게시 |
| Q&A 자동 응답 | 새 질문 감지 → AI 답변 제안 |
| 경쟁사 분석 | Google Maps 경쟁사 비교 |

**고객 가치:** Calls/Directions 증가, 노출 상승, 시간 절약

### ⭐ CORE 2. AI 콘텐츠 생성 & 멀티채널 자동 업로드

| 플랫폼 | 지원 |
|--------|------|
| Google Business | ✅ 필수 |
| Instagram | ✅ 필수 |
| Threads | ✅ |
| Facebook Page | 옵션 |

| 기능 | 설명 |
|------|------|
| AI 주제 추천 | 업종/지역/날씨/시즌 기반 |
| AI 텍스트 생성 | GPT/Gemini 기반 |
| AI 이미지 생성 | Gemini 3 Pro Image |
| 해시태그 최적화 | 자동 생성 |
| 예약 포스팅 | Scheduler |

**고객 가치:** 주 3-5시간 절약, 꾸준한 활동, 브랜딩 유지

### ⭐ CORE 5. Missed Call Text Back (부재중 전화 자동 문자)

| 기능 | 설명 |
|------|------|
| 전화 포워딩 | Twilio 번호 → 실제 번호 연결 |
| 부재중 감지 | busy/no-answer/failed 감지 |
